# 前言

## 本书的使命

AgenticWiki 是一个 Agent 驱动的前端代码 → Wiki 转换系统。本书旨在系统性地拆解这个项目的完整工作流，帮助读者从"知道怎么用"到"理解为什么这样设计"。

## 读者对象

- **新加入项目的开发者**：快速了解流水线全貌，理解每个阶段的设计意图
- **维护者**：深入理解每个脚本的策略和实现，便于排查问题和扩展功能
- **架构爱好者**：了解一个复杂的 Agent 编排系统如何分层解耦、如何处理边界情况

## 核心原则

**Agent 只需做两件事**：
1. 执行 `npx tsx src/runner.ts --project <path>`
2. 在 GEN 阶段暂停时 `spawn_agent` SubAgent

其余所有工作——扫描、依赖分析、调度、组装、验证——全部由 Runner 自动化完成。
